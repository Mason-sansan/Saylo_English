# GStack Design HTML (Cursor)

## Goal
Ship-ready HTML/CSS with computed layout behavior (text reflow, resilient spacing), not a one-viewport mock.

## Constraints (taste)
- No gradients, no emojis, no box-shadows, no decorative borders.
- Prefer open layout over card walls.
- One primary action per screen.
- Typography calm; avoid oversized headings.

## Deliverable
One HTML file with embedded CSS, mobile-first, readable without JavaScript.

## Checklist
- Works at 360px width and scales to 440–560px cleanly.
- Copy is concise and non-repetitive.
- Visual hierarchy is obvious within 2 seconds.
