# GStack Plan Design Review (Cursor)

## Purpose
Enforce taste and restraint: remove generic learning-app patterns, keep outputs curated.

## What to check
- **Information hierarchy**: what is the one thing that stands out?
- **Flow friction**: number of choices, number of screens, time to start
- **Copy**: calm, concise, slightly cold; no praise/no fluff
- **Output quality**: diagnostic card must be skimmable and non-repetitive
- **Anti-pattern scan**: gamification, progress theater, verbose explanations

## Output format
- **Top issue**: 1 sentence
- **3 cuts**: remove/merge/simplify
- **2 upgrades**: concrete improvements
- **Final spec**: 6 bullets describing the interaction exactly
