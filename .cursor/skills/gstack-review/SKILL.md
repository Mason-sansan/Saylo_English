# GStack Review (Cursor)

## Focus
- Correctness and failure modes
- Scope control (avoid drive-by edits)
- Simplicity (remove cleverness)
- UX slop detection (generic patterns, verbose output)

## Output format
- **Must fix**: 1–5 bullets
- **Should fix**: 1–5 bullets
- **Nice to have**: 0–3 bullets
- **Risk**: 1 sentence
